# Online Demo Deploy Package

This folder is ready for GitHub + Cloudflare Pages.

## Files included

- index.html
- styles.css
- app.js
- data/worldcup-data.js
- README.md
- README_EN.md
- DISCLAIMER.md

## Cloudflare Pages settings

- Framework preset: None
- Build command: leave empty
- Build output directory: `/` or leave empty

## Steps

1. Create a new GitHub repository.
2. Upload everything inside this folder to the repository root.
3. Go to Cloudflare Pages.
4. Create a new project.
5. Connect your GitHub repository.
6. Deploy.
7. Open the generated `.pages.dev` URL.

## Note

This online demo is a static fan-analysis tool. It does not auto-update live data and is not betting advice.
